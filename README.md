<h1 align="center">StockFlow - Sistema de Gestão de Materiais e Estoque</h1>

<h3 align="center">Sobre o Projeto</h3>

<p align="left-center">O StockFlow é um sistema de gerenciamentode estoque de materiais e controle de estoque desenvolvido em Java, com o objetivo de simular funcionalidades presentes em sistemas corporativos de gestão (ERP).</p>

<p align="left-center">O projeto foi criado no contexto acadêmico, inspirado em rotinas reais de controle logístico e produtivo, permitindo o acompanhamento de materiais, estrutura de produtos (BOM), movimentações de estoque e análise de perdas (scrap).</p>

<h3 align="center">Objetivo</h3>

<p align="left-center">Fornecer uma aplicação capaz de centralizar e organizar informações relacionadas á gestão de materiais, permitindo maior controle operacional e apoio á tomada de decisão.</p>

<h3 align="center">Funcionalidades</h3>

- Cadastro e consulta de Materiais
- Atualização de custo de materiais
- Controle de estoque
- Gestão de produtos e estrutura BOM (Bill of Materials)
- Registro e análise de scrap
- Navegação dinâmica entre telas com CardLayout

<h3 align="center">Tecnologias Utilizadas</h3>

- Java
- Java Swing (Interface Gráfica)
- POO
